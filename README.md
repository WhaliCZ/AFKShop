# AFKShop
# AFKShop  A Paper Minecraft plugin that allows players to open AFK shops in a designated marketplace region.  Players become invisible and are represented by a custom ArmorStand model while their shop is active.  ## Features - Players open a shop with `/afkshop` and are frozen in place - Custom GUI editor (54 slots) for placing items and setting prices - Buyers can right-click the shop owner's model to browse and purchase items - Earnings are tracked and paid out when the shop closes - SQLite database for persistent earnings storage - Custom region system (no WorldGuard required) - Vault economy integration  ## Requirements - Paper 1.21+ - Java 17+ - Vault + an economy plugin (e.g. EssentialsX)  ## Commands | Command | Description | |---|---| | `/afkshop` | Open your AFK shop | | `/afkshop stop` | Close your shop and receive earnings | | `/afkshop setregion pos1` | Set first corner of the AFK market region | | `/afkshop setregion pos2` | Set second corner of the AFK market region | | `/afkshop setregion info` | Show current region coordinates |  ## Permissions | Permission | Description | Default | |---|---|---| | `afkshop.use` | Open an AFK shop | true | | `afkshop.admin` | Set region, break shops | op |  ## Setup 1. Drop the jar into your `plugins/` folder 2. Start the server 3. Go to the area you want as the AFK market 4. Run `/afkshop setregion pos1` at one corner 5. Run `/afkshop setregion pos2` at the opposite corner 6. Players can now use `/afkshop` inside the region  ## How it works 1. Player runs `/afkshop` inside the market region 2. A shop editor GUI opens — drag items into slots 3. **Shift+click** an item to set its price via chat 4. Other players right-click the shop owner's model to open the buyer GUI 5. The owner runs `/afkshop stop` to close the shop and collect earnings
